# ✅ **1. How to use Google Maps in React**

There are several ways to integrate Google Maps in a React project:

---

### **Option 1: Using `google-maps-react` library**

1. **Install the library**:

```bash
npm install google-maps-react
```

2. **Get a Google Maps API key**:

   * Go to [Google Cloud Console](https://console.cloud.google.com/).
   * Enable **Maps JavaScript API**.
   * Generate an API key.

3. **Basic usage**:

```jsx
import React from 'react';
import { Map, GoogleApiWrapper, Marker } from 'google-maps-react';

const MapContainer = (props) => {
    return (
        <Map
            google={props.google}
            zoom={14}
            initialCenter={{ lat: 23.8103, lng: 90.4125 }} // Example: Dhaka
        >
            <Marker position={{ lat: 23.8103, lng: 90.4125 }} />
        </Map>
    );
};

export default GoogleApiWrapper({
    apiKey: 'YOUR_GOOGLE_MAPS_API_KEY'
})(MapContainer);
```

* `Map` → Displays the map
* `Marker` → Adds a marker at a specific position
* `initialCenter` → Sets the initial coordinates of the map
* `zoom` → Sets the zoom level

---

### **Option 2: Using `@react-google-maps/api` (recommended for modern React)**

1. **Install the library**:

```bash
npm install @react-google-maps/api
```

2. **Basic usage**:

```jsx
import React from 'react';
import { GoogleMap, LoadScript, Marker } from '@react-google-maps/api';

const containerStyle = {
    width: '100%',
    height: '400px'
};

const center = {
    lat: 23.8103,
    lng: 90.4125
};

const MyMap = () => {
    return (
        <LoadScript googleMapsApiKey="YOUR_API_KEY">
            <GoogleMap
                mapContainerStyle={containerStyle}
                center={center}
                zoom={14}
            >
                <Marker position={center} />
            </GoogleMap>
        </LoadScript>
    );
};

export default MyMap;
```

* `LoadScript` → Loads the Google Maps JS API
* `GoogleMap` → Displays the map
* `Marker` → Adds a marker
* `mapContainerStyle` → Defines map container size

# ✅ **2. Common Functions/Features You Can Implement**

Here are common things you can do with Google Maps in a React project:

| Feature / Function       | Description                                                     |
| ------------------------ | --------------------------------------------------------------- |
| **Marker**               | Place a marker at a specific location on the map.               |
| **InfoWindow**           | Show popup information when a marker is clicked.                |
| **Map Centering**        | Center the map to a specific latitude & longitude.              |
| **Zoom Control**         | Adjust the zoom level programmatically or via user interaction. |
| **Polyline / Polygon**   | Draw lines, routes, or areas on the map.                        |
| **Geolocation**          | Get user’s current location using `navigator.geolocation`.      |
| **Directions / Routing** | Show routes between two points using `DirectionsService`.       |
| **Map Events**           | Listen to events like `onClick`, `onDrag`, `onZoomChanged`.     |
| **Custom Marker Icons**  | Use custom images/icons for markers.                            |
| **Cluster Markers**      | Group multiple markers when they are close to each other.       |
| **Street View**          | Display panoramic street view images.                           |
| **Map Styling**          | Customize the map appearance (colors, roads, labels).           |

# ✅ **3. Tips for React Projects**

1. **Use `@react-google-maps/api`** for modern React — better performance and hooks support.
2. Keep the **API key secret** — do not commit it to GitHub. Use `.env` file.
3. Start with **Marker and Map Center** first, then add advanced features like routing or polygons.
4. Use **state** and **props** to dynamically update markers, zoom, or center.
