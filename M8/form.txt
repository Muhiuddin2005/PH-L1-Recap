# âœ… 1. What is the `<form>` tag in React?

In React, the `<form>` tag is the same HTML `<form>` element, but React often manages form data using **state** and **event handlers** (e.g., `onSubmit`, `onChange`).

You write it like normal HTML:

```jsx
<form onSubmit={handleSubmit}>
  <input type="text" />
  <button type="submit">Submit</button>
</form>
```

React does **not** replace the `<form>` tagâ€”it just adds better ways to control form data.

# âœ… 3. What is the `<input>` tag in React?

`<input>` in React is also the same as normal HTML, but React often uses **state** to track the value.

Example:

```jsx
<input 
  type="text"
  value={username}
  onChange={(e) => setUsername(e.target.value)}
/>
```

This is called a **controlled input**.
