# ğŸ“Œ **Step 1: Create a JS Utility File**

Create a file:

### ğŸ‘‰ `mathFunctions.js`

```js
// Basic arithmetic functions
function add(a, b) {
  return a + b;
}

function subtract(a, b) {
  return a - b;
}

function multiply(a, b) {
  return a * b;
}

function divide(a, b) {
  return a / b;
}

// Export using "as" keyword
export {
  add as addNumbers,
  subtract as subtractNumbers,
  multiply as multiplyNumbers,
  divide as divideNumbers
};
```

# ğŸ“Œ **Step 2: Import and Use in a React Component**

**src/App.jsx**

### ğŸ‘‰ `App.jsx`

```jsx

  addNumbers,
  subtractNumbers,
  multiplyNumbers,
  divideNumbers
} from "./utils/mathFunctions";

export default function App() {
  const a = 10;
  const b = 5;

  return (
    <div>
      <h1>Arithmetic Demo</h1>

      <p>Add: {addNumbers(a, b)}</p>
      <p>Subtract: {subtractNumbers(a, b)}</p>
      <p>Multiply: {multiplyNumbers(a, b)}</p>
      <p>Divide: {divideNumbers(a, b)}</p>
    </div>
  );
}
```

| Feature / Point                    | **Default Export**                | **Named Export**                          |
| ---------------------------------- | --------------------------------- | ----------------------------------------- |
| **How many per file?**             | Only **1** allowed                | **Unlimited** (you can export many items) |
| **Export syntax**                  | `export default funcName;`        | `export { funcName1, funcName2 };`        |
| **Import syntax**                  | `import Something from "./file";` | `import { funcName1 } from "./file";`     |
| **Can rename while importing?**    | Yes, freely                       | Yes, but must use `as`                    |
| **Best for**                       | Components, classes, configs      | Utility functions, helpers, constants     |
| **Clarity**                        | Less clear (name can be anything) | Very clear (must match exported name)     |
| **Autocompletion reliability**     | Medium (name can change anywhere) | Strong (editor suggests all exports)      |
| **Can you export multiple items?** | âŒ No                              | âœ… Yes                                     |
| **Recommended for utility files?** | âŒ Not recommended                 | âœ… Perfect choice                          |
| **Example use case**               | `export default App;`             | `export { add, subtract, multiply }`      |
