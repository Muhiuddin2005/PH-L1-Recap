# ‚úÖ **What does `result` and `result.user` return in Firebase Authentication?**

When you use:

```js
signInWithPopup(auth, googleProvider)
  .then(result => { ... })
```

### ‚úî `result` = **Full authentication response object**

Firebase returns a **credential + user info + extra data**.

### ‚úî `result.user` = **The logged-in user's information**

This is the **Firebase User object**, containing personal details like name, email, photo, etc.

---

# üéØ **What information can you get from `result.user`? (All useful fields)**

Here is the *most complete and beginner-friendly list*.

---

## ‚úÖ **1. Basic User Info**

| Property           | Example                              | Meaning                                |
| ------------------ | ------------------------------------ | -------------------------------------- |
| `user.uid`         | `"39qdk2XYZ..."`                     | Unique ID for this user                |
| `user.displayName` | `"John Doe"`                         | Full name from Google                  |
| `user.email`       | `"john@gmail.com"`                   | Email address                          |
| `user.photoURL`    | `"https://lh3.googleusercontent..."` | Profile image                          |
| `user.phoneNumber` | `null`                               | Phone number (if logged in with phone) |
| `user.providerId`  | `"firebase"`                         | Main provider                          |

---

## ‚úÖ **2. Login Provider Details**

Use:

```js
user.providerData[0]
```

This gives deeper provider information.

| Property      | Meaning                           |
| ------------- | --------------------------------- |
| `providerId`  | `"google.com"`                    |
| `uid`         | Provider-specific ID (Google UID) |
| `displayName` | Google name again                 |
| `email`       | Google email again                |
| `photoURL`    | Profile picture again             |
| `phoneNumber` | If available                      |

---

## ‚úÖ **3. Account Metadata**

These are available as:

```js
user.metadata
```

| Property         | Meaning                               |
| ---------------- | ------------------------------------- |
| `creationTime`   | The time the user account was created |
| `lastSignInTime` | Last login time                       |

---

## ‚úÖ **4. Access Token / Credential Info**

Located inside `result` (not `result.user`):

```js
const credential = GoogleAuthProvider.credentialFromResult(result);
const token = credential.accessToken;
```

| Property      | Meaning               |
| ------------- | --------------------- |
| `accessToken` | Google OAuth token    |
| `idToken`     | Google ID token (JWT) |

---

## ‚úÖ **5. Email Verification Info**

```js
user.emailVerified
```

| Value   | Meaning        |
| ------- | -------------- |
| `true`  | Email verified |
| `false` | Not verified   |

---

# üìå Example: Print All User Info

```js
console.log("UID:", user.uid);
console.log("Name:", user.displayName);
console.log("Email:", user.email);
console.log("Photo:", user.photoURL);
console.log("Verified:", user.emailVerified);

console.log("Created At:", user.metadata.creationTime);
console.log("Last Login:", user.metadata.lastSignInTime);

console.log("Provider:", user.providerData[0].providerId);
console.log("Google UID:", user.providerData[0].uid);
```
