# âœ… **1. What is `e.target.reset()`?**

When you submit a form:

```jsx
const handleSubmit = (e) => {
  e.preventDefault();
  e.target.reset();
};
```

### âœ” `e.target` = the `<form>` itself

### âœ” `reset()` = clears all input fields

So it works like:

* Clears name field
* Clears email
* Clears password
* Resets checkboxes, etc.

It is NOT mandatory â€” only used if you want to clear the form after submit.

# âœ… **2. Email Verification in Firebase**

Firebase sends a verification email using:

```js
import { sendEmailVerification } from "firebase/auth";
```

After user registers:

```js
createUserWithEmailAndPassword(auth, email, password)
  .then(result => {
      const user = result.user;

      // Send verification email
      sendEmailVerification(user)
        .then(() => {
          setSuccess("Verification email sent! Check inbox.");
        });
  })
  .catch(error => {
      setError(error.message);
  });
```

# âœ… **3. Block Login if Email Not Verified**

During login:

```js
signInWithEmailAndPassword(auth, email, password)
  .then(result => {
      const user = result.user;

      if (!user.emailVerified) {
        setError("Your email is not verified. Please check inbox.");
        signOut(auth);  // ðŸ”¥ important
        return;
      }

      setSuccess("Login successful!");
  })
  .catch(error => setError(error.message));
```

ðŸ” **Important:**

* Firebase stores `emailVerified: false` until user clicks the verification link.
* You must call `signOut` to prevent unverified users from staying logged in.

# âœ… **4. Reset Password using Firebase + emailRef**

You must import:

```js
import { sendPasswordResetEmail } from "firebase/auth";
```

### Use `useRef` hook:

```js
const emailRef = useRef();
```

### Reset Password Handler:

```js
const handleResetPassword = () => {
  const email = emailRef.current.value;

  if (!email) {
    setError("Please enter your email first");
    return;
  }

  sendPasswordResetEmail(auth, email)
    .then(() => setSuccess("Reset email sent! Check inbox."))
    .catch(error => setError(error.message));
};
```
